# iDox.ai Guardrail

Real-time AI guardrails that prevent sensitive data exposure, AI agent misuse, and unauthorized file access.

## 🚨 Why Guardrail
AI tools and agents can access and leak sensitive data (e.g., OpenClaw risks). Guardrail stops this before it happens.

## 🔐 Key Capabilities
- Real-time prompt and response monitoring
- Sensitive data detection (PII, PHI, financial, IP)
- File access control for AI agents
- Data redaction and anonymization
- Endpoint-level enforcement
